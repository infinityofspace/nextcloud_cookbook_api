About
+++++

This is a Python API client for the Nextcloud Cookbook app API version `0.1.3`. It allows you to interact with your
Nextcloud Cookbook instance programmatically, enabling you to manage recipes, ingredients, and other related data.
You can find the official documentation of the Nextcloud Cookbook API
`here <https://nextcloud.github.io/cookbook/dev/api/0.1.3/index.html>`_.

Link to the source code: `Github <https://github.com/infinityofspace/nextcloud_cookbook_api>`_
